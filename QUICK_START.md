# 建築会社スタッフアプリ - クイックスタート

## すぐに動かす

```bash
cd /Users/natsuki.akiyama/construction-staff-app
npm install
npm run dev
```

ブラウザで `http://localhost:3000` を開く

## 画面説明

### トップページ (`/`)
- **ダッシュボードを開く**: タブ切り替え式の統合画面（推奨）
- **従業員ダッシュボード**: 従業員用の単独画面
- **マネージャー画面**: 管理者用の単独画面

### 統合ダッシュボード (`/demo/dashboard`) ⭐ 推奨

**ヘッダーのタブで従業員/マネージャーを切り替えられます**

#### 👷 従業員タブ

1. **ウェルカムバナー**
   - 従業員名が表示されます

2. **コンディションチェック**
   - 5つの質問に1〜5で回答
   - 相談事項を入力（任意）
   - 「記録する」ボタンで保存

3. **最近のコンディション**
   - 過去のコンディション記録が表示されます

4. **あなたの評価**
   - 評価サマリー: 受け取った評価タグと回数
   - タイムライン: 時系列で評価を確認

5. **親方の予定（カレンダー形式）**
   - 月のカレンダーで予定を確認
   - 今日の予定詳細も表示

6. **掲示板（事務連絡）**
   - 重要なお知らせは📍ピン留め
   - 「すべて見る」で全件表示

7. **評価・感謝を送る（右下のボタン）**
   - 他の従業員に感謝やスキル評価を送信
   - タグとコメントで気持ちを伝える

#### 👔 マネージャータブ

1. **サマリーカード**
   - 提出済み人数、平均スコア、要注意人数

2. **従業員コンディション一覧**
   - 全従業員のコンディションを一覧表示
   - スコアに応じて色分け（赤: 要注意、緑: 良好）

3. **詳細モーダル**
   - 従業員をクリックすると詳細表示
   - 各質問への回答を確認
   - 相談事項を確認
   - **AIアドバイス**: 状況に応じた対応提案

### 個別画面

- **従業員ダッシュボード**: `/demo/employee` - 従業員ビューのみ
- **マネージャー画面**: `/demo/manager` - マネージャービューのみ
