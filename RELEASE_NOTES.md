# 🎉 v1.1.0 リリースノート

## 2026年6月3日リリース

### ✨ 新機能ハイライト

#### 1. クイックアクセスバー（画面下部固定）

スクロールせずに、いつでも親方の予定と掲示板にアクセスできるようになりました！

**特徴**:
- 画面下部に常時表示
- 大きなボタンでタップしやすい
- 📅 親方の予定 / 📌 掲示板

#### 2. カレンダーモーダル（日付タップで予定表示）

カレンダーの日付をタップすると、その日の予定が表示されます。

**機能**:
- 月間カレンダー表示
- 予定がある日には青いドット
- 日付タップで詳細表示（時間・タイトル・場所）
- 視覚的にわかりやすいデザイン

**使い方**:
1. 画面下部の「📅 親方の予定」をタップ
2. カレンダーが開く
3. 見たい日付をタップ
4. その日の予定が表示される

#### 3. 事務連絡投稿フォーム ⭐

誰でも簡単に事務連絡を投稿できるようになりました！

**機能**:
- タイトル入力（50文字まで）
- 内容入力（300文字まで）
- 📍 ピン留めオプション（重要なお知らせ）
- リアルタイムプレビュー
- 投稿後にチーム全員に通知

**使い方**:
1. 画面下部の「📌 掲示板」をタップ
2. 「✍️ 新しい連絡を投稿」ボタンをタップ
3. タイトルと内容を入力
4. 必要に応じて「固定する」にチェック
5. 「投稿する」ボタンをタップ

### 🔧 改善点

- **スクロール量の削減**: 親方の予定と掲示板をモーダル化
- **アクセス性向上**: 固定ボタンでいつでもアクセス可能
- **使いやすさ向上**: 日付タップで予定確認が直感的に
