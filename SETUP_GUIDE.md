# セットアップガイド

## 前提条件

- Node.js 20以上がインストールされていること

## インストール手順

### 1. プロジェクトディレクトリに移動

```bash
cd /Users/natsuki.akiyama/construction-staff-app
```

### 2. 依存関係のインストール

```bash
npm install
```

### 3. 開発サーバーの起動

```bash
npm run dev
```

### 4. ブラウザでアクセス

`http://localhost:3000` を開いてください。

トップページに「従業員ダッシュボード」のボタンが表示されます。

## 動作確認

### 従業員ダッシュボード画面

`http://localhost:3000/demo/employee` にアクセスすると、以下の機能を確認できます:

1. **ウェルカムバナー**: 従業員名が表示されます
2. **コンディションチェック**: 5つの質問に1-5で回答し、相談事項を入力できます
3. **最近のコンディション**: 過去のコンディション記録が表示されます
4. **親方の予定**: 今後の予定が一覧表示されます
5. **掲示板**: 事務連絡が表示されます

## トラブルシューティング

### ポート3000が使用中の場合

別のポートで起動:
```bash
npm run dev -- -p 3001
```

### npm installでエラーが出る場合

キャッシュをクリア:
```bash
npm cache clean --force
npm install
```
