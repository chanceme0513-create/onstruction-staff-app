# 建築会社スタッフアプリ - 機能詳細

## 感謝文化を育てる設計 💚

このアプリは「感謝を送る文化」を定着させるために、以下の仕組みを実装しています。

### 1. 感謝の可視化・動機づけ

#### 🎯 今日の感謝チャレンジ
- **目的**: 毎日の感謝目標を設定し、達成感を与える
- **表示**: 画面上部に大きく表示（見逃せない）
- **機能**:
  - 今日送った感謝の数 / 目標数
  - プログレスバーで視覚的に進捗を確認
  - 達成時に「+10ポイント獲得」などの報酬表示
- **動機**: 「あと1人で目標達成！」と具体的な行動を促す

#### 🏆 今週の感謝ランキング
- **目的**: ゲーミフィケーションで競争心を刺激
- **機能**:
  - 感謝を送った回数でランキング
  - 自分の順位をハイライト表示
  - 「あと〇回で1位！」と具体的な目標提示
  - 1位・2位・3位に金銀銅のメダル表示
- **動機**: 上位を目指すことで自然と感謝が増える

#### ⚡ クイック感謝
- **目的**: 感謝送信の障壁を極限まで下げる
- **機能**:
  - タップ2回で送信完了
  - ステップ1: メンバーを選ぶ
  - ステップ2: 感謝タグを選ぶ
  - 送信後に「🎉送信しました！」と気持ちよいフィードバック
- **動機**: 「簡単だからやってみよう」→習慣化

#### 💬 最近のチーム感謝
- **目的**: 他の人が感謝を送っている様子を見せる（社会的証明）
- **機能**:
  - リアルタイムで感謝タイムラインを表示
  - 「誰が誰に何の感謝を送ったか」を可視化
  - 「今日はチーム全体で32回の感謝が送られました」と総数表示
- **動機**: 「みんなやってるから自分も」という心理

### 2. チームの雰囲気可視化

#### 🏗️ チームの雰囲気
- **目的**: チーム全体の状態を把握し、助け合いを促進
- **機能**:
  - 全員のコンディション平均スコアを表示
  - スコアに応じて絵文字と色が変化
  - 報告済み人数を表示
- **効果**: 「チームの調子が悪そう→励ましの言葉を送ろう」

### 3. 今日の現場情報

#### 📍 今日の現場情報
- **機能**:
  - 天気・気温
  - 稼働中の現場数
  - 出勤人数
  - 進捗状況
  - 注意喚起（熱中症対策など）
- **効果**: チーム全体の状況を共有し、連帯感を生む

### 4. 従業員ポートフォリオ

#### ⭐ あなたの評価
- **目的**: 自分が受け取った感謝を確認し、モチベーションアップ
- **機能**:
  - **評価サマリー**: どの評価タグを何回受け取ったか
  - **タイムライン**: 時系列で感謝・評価を確認
  - コメント付きで温かみのあるフィードバック
- **効果**: 「自分も誰かに感謝を送ろう」という気持ちになる

## 感謝文化が定着する理由

### 心理的な仕組み

1. **目標の明確化**
   - 「今日3人に感謝を送る」という具体的な目標
   - 進捗バーで達成度が視覚的にわかる

2. **即座のフィードバック**
   - 送信直後に「🎉送信しました！」
   - ポイント獲得など、報酬を即座に提示

3. **社会的証明**
   - 「みんなが感謝を送っている」タイムラインで可視化
   - 「自分もやらなきゃ」という心理が働く

4. **ゲーミフィケーション**
   - ランキングで競争心を刺激
   - 「もう少しで1位」という気持ちにさせる

5. **障壁の低減**
   - クイック感謝はタップ2回で完了
   - フローティングボタンは削除し、常に見える場所に配置

6. **相互作用の促進**
   - 感謝を受け取ると嬉しい
   - 嬉しいから自分も送りたくなる
   - ポジティブなループが生まれる
