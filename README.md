# 建築会社スタッフアプリ

建築会社向けの従業員コミュニケーション・コンディション管理アプリです。

従業員が日々のコンディションを報告し、親方の予定を確認し、事務連絡を共有できる場を提供します。

---

## 主な機能

- **コンディション報告**: 5問のアンケート形式で日々の状態を記録 + 実務的な相談を書き込み
- **カレンダー**: 親方の予定を確認できる
- **掲示板**: 事務連絡をチーム全員で共有
- **ダッシュボード**: 自分のコンディション履歴を確認

---

## セットアップ

### 前提条件

- Node.js 20+
- npm（または pnpm）

### 1. 依存関係のインストール

```bash
cd construction-staff-app
npm install
```

### 2. 開発サーバー起動

```bash
npm run dev
```

`http://localhost:3000` でアクセスできます。
